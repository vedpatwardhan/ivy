Elementwise functions¶

ivy.functional.ivy.array_api.elementwise_functions.asinh(x: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Calculates an implementation-dependent approximation to the inverse hyperbolic sine, having domain [-infinity, +infinity] and codomain [-infinity, +infinity], for each element x_i in the input array x.

Special cases For floating-point operands, - If x_i is NaN, the result is NaN. - If x_i is +0, the result is +0. - If x_i is -0, the result is -0. - If x_i is +infinity, the result is +infinity. - If x_i is -infinity, the result is -infinity.

Parameters

x – input array whose elements each represent the area of a hyperbolic sector. Should have a floating-point data type.

Returns

an array containing the inverse hyperbolic sine of each element in x. The returned array must have a floating-point data type determined by type-promotion.

ivy.functional.ivy.array_api.elementwise_functions.bitwise_and(x1: Union[ivy.array.Array, ivy.NativeArray], x2: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Computes the bitwise AND of the underlying binary representation of each element x1_i of the input array x1 with the respective element x2_i of the input array x2.

Parameters

  • x1 – first input array. Should have an integer or boolean data type.

  • x2 – second input array. Must be compatible with x1 (see Broadcasting). Should have an integer or boolean data type.

Returns

an array containing the element-wise results. The returned array must have a data type determined by Type Promotion Rules.

ivy.functional.ivy.array_api.elementwise_functions.isfinite(x: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Tests each element x_i of the input array x to determine if finite (i.e., not NaN and not equal to positive or negative infinity).

Parameters

x – input array. Should have a numeric data type.

Returns

an array containing test results. An element out_i is True if x_i is finite and False otherwise. The returned array must have a data type of bool.

ivy.functional.ivy.array_api.elementwise_functions.sqrt(x: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Calculates the square root, having domain [0, +infinity] and codomain [0, +infinity], for each element x_i of the input array x. After rounding, each result must be indistinguishable from the infinitely precise result (as required

by IEEE 754).

param x

input array. Should have a floating-point data type.

return

an array containing the square root of each element in x. The returned array must have a floating-point

data type determined by Type Promotion Rules.
